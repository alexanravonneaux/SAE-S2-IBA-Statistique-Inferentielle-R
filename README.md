## 📋 Description

Application de la statistique inférentielle avec R : tests d'hypothèses (Student, Chi², ANOVA), intervalles de confiance et calcul de p-values. Ce projet contextualise les tests dans un cadre IBA (Indicateurs de Business Activité) pour tirer des conclusions sur des populations à partir d'échantillons.

## 🎯 Objectifs pédagogiques
- Formuler et tester des hypothèses H0/H1
- Réaliser des tests paramétriques (t-test, ANOVA) et non-paramétriques (Chi²)
- Calculer et interpréter des intervalles de confiance
- Conclure statistiquement avec le niveau de signification α

## 🔧 Technologies

| Outil | Rôle |
|-------|------|
| R | Tests statistiques (t.test, aov, chisq.test) |
| dplyr | Préparation des données |
| ggplot2 | Visualisation des distributions et tests |

## 📊 Résultats
- Tests t de Student uni/bilatéraux réalisés et interprétés
- ANOVA à un facteur avec test post-hoc de Tukey
- Intervalles de confiance à 95% calculés et visualisés
